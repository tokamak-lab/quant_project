# FI-2010 Dataset

This project uses the public FI-2010 dataset for limit order book prediction.

## Download
- Source: https://etsin.fairdata.fi/dataset/73eb48d7-4dbc-4a10-a52a-da745b47a649
- Download and extract the dataset locally.

## Expected layout
Place the FI-2010 text files under:

```
data/fi2010/
  Train_Dst_NoAuction_ZScore_CF_1.txt
  Test_Dst_NoAuction_ZScore_CF_1.txt
  ...
```
